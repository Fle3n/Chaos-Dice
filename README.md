# Chaos-Dice
Chaos Dice website, created fle3n (Zaynutdin) and TNT_GAMER_303 (Kadirov)

Установите зависимости через команду pip install -r requirements.txt
или же альтернитивный вариант: pip install Flask Flask-SocketIO

затем запустите файл app.py и подключитесь к http://localhost:5001 (т.к. 5000 порт занят у маков, пришлось выбрать 5001 а не стандартный 5000)

У сайта есть мобильная версия (при изменении размеров экрана сайт будет принимать наиболее удобный размер иконок, и их позицию)

Как работает сайт:
В самом начале нас встречает регистрация/авторизация, данные сохраняются в базе данных с зашифрованным паролем, обеспечивая безопасность.

В главном меню есть глобальный чат между игроками слева.
По центру можно бросать кубики в одиночном режиме, выбрав размер кубика, и кол-во бросков, тут сохраняется история бросков, есть красивая анимация и имеется подсчет суммы кубиков.

Сверху есть вкладки:
Главная - ведет сюда же
Разработчики - ведет на страницу где указано о разработчиках которые делали этот проект
Лидеры - страница с лидербордом, по каким то статистикам
Профиль - можно выйти из него, а также посмотреть личную статистику

Справа вкладка мультиплеера, что бы его проверить запустите сайт с двух и более браузеров, тут можно создать комнату, выбрать будет ли она приватная (если она приватная она не отображается в списках доступных комнат, и надо будет зайти по коду и ввести пароль который вы указываете при создании комнаты), можно подключится по коду, или же в списке доступных комнат если она открытая.

О мультиплеере:
размер комнаты до 4 людей, для начала игры нужно как минимум 2 игрока, на странице комнаты видно ее название и код комнаты, слева видно список подключенных пользователей а также таблицу лидеров. Справа есть чат комнаты, чуть ниже чата есть история раундов.
Как работает игра в мультиплеера:
Суть игры при броске кубиков выбить больше сумму очков, чем твой/твои противники, кол-во кубиков и их граней выбирает создатель комнаты, когда он начинает раунд, по очереди каждый пользователь кидает кубики, если он не кидает в течении 7 секунд (таймер идет на главной странице) за него кубики кидаются автоматически, пользователи в реальном времени видят как каждый кидает кубики, и видит сумму его очков. Если создатель комнаты выходит из нее, следующим создателем комнаты становится другой пользователь который подключился раньше остальных.
